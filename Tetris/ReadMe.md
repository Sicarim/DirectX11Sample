# DirectX11 기반으로 테트리스 게임 개발

### 게임 방법

* 좌/우 방향키로 이동 블록을 좌/우로 이동할 수 있다.
* 상 방향키로 이동 블록을 회전시킬 수 있다.
* 하 방향키로 이동 블록을 2 개 블록 높이만큼 아래로 이동할 수 있다.

### 알려진 버그

* 블록이 좌/우 벽과 가까이 있을 경우 회전이 되지 않을 수 있습니다.
  * 회전시 좌/우 벽과 충돌된다고 판단되면 회전하지 못 하도록 개발하였습니다.
